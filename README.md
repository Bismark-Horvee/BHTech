This project is a simple educational website called BHTech.
It includes a navigation bar, hero section, courses section, admissions form, about section, login modal, and a print button.

Files Included

index.html – Main structure of the website

horvee.css – Styling for the website

horvee.js – JavaScript functionality

Features

Responsive header and navigation

Hero section with animations

Course search and listing area

Multi-step admission form

Login popup modal

Print button

Smooth UI/UX interactions

How to Use

Download or clone the project.

Open index.html in any browser.

Make sure horvee.css and horvee.js are in the same folder.

Technologies Used

HTML

CSS
